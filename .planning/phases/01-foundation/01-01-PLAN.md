       - Define CampaignStatus(str, Enum) with values: pending, sourcing, enriching, pushing, complete, failed
       - Define ContactStage(str, Enum) with values: sourced, enriched, pushed
       - Define Campaign(SQLModel, table=True): id UUID default_factory=uuid4, name str, target_criteria str (store as JSON string), status CampaignStatus default pending, created_at datetime default utcnow, updated_at datetime default utcnow
       - Define Contact(SQLModel, table=True): id UUID default_factory=uuid4, campaign_id UUID Field(foreign_key="campaign.id"), email str, first_name/last_name/title/company/linkedin_url all Optional[str] default None, verified bool default False, stage ContactStage default sourced, created_at datetime default utcnow
       - Define PipelineLog(SQLModel, table=True): id Optional[int] Field(default=None, primary_key=True), campaign_id UUID Field(foreign_key="campaign.id"), stage str, status str, message Optional[str] default None, timestamp datetime default utcnow
